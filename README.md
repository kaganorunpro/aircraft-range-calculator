# Aircraft Range Calculator ✈️

A simple Python project that estimates the maximum flight range of an aircraft.

## Features

- Calculates aircraft range
- Evaluates aircraft load
- Determines mission category
- Uses user input
- Uses if-elif-else decision structures

## Inputs

- Fuel Capacity (Liters)
- Fuel Consumption (L/100 km)
- Number of Passengers

## Example

Fuel Capacity: 300 L

Fuel Consumption: 15 L/100 km

Passengers: 4

Output:

Maximum Range: 2000 km

Aircraft Load: Medium

Mission Status: MEDIUM RANGE

## Technologies

- Python
